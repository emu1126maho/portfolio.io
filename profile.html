わたしのこと

Mochizuki Maho

望月 真帆

1998年生まれ。長崎県出身。

情報やブランドイメージを動きのある形で伝えることができるWebデザインに惹かれ、
Webデザイナーを目指すことを決意し、創造社リカレントスクール福岡校に入校。
誰に対しても分かりやすく使いやすいデザインを心掛けており、
細部にまで意味を持ったデザインにすることによって、
ユーザーの心を動かすようなWebデザイナーを目指しています。

いまできること

Web design

Webサイト、バナーの制作をすることができます。ユーザーが操作に迷うことのないシンプルでわかりやすいデザインを心がけながら、 些細なところまで丁寧に作り込むようにしています。足りないパーツをイラストやグラフィックで補うことも可能です。

スキルを詳しくみる

  • Illustrator

    名刺・チラシ・バナーなどの制作ができます

  • Photoshop

    画像の補正、切り抜き、簡単なバナーの作成をすることができます

  • Figma

    Webデザイン制作はFigmaを使用しています
    プロトタイプ機能を使い、動きのあるWebサイトを事前に再現することができます

Web development

デザインカンプを忠実に再現することができます。規則性のある命名をし、誰が見てもわかりやすいコードを書くことを心がけています。 レスポンシブ対応も可能です。AIを使用して動きをつけることができます。

スキルを詳しくみる

  • HTML / CSS

    規則性のあるコーディングを行うことができます
    使用しているエディターはVisual Studio Codeです

  • JavaScript

    jQueryを使用して動きを加えることができます
